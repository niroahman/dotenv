# dotenv

Personal macOS dotfiles. Symlink-based, idempotent.

## Quickstart

```bash
git clone git@github.com:niroahman/dotenv.git
cd dotenv && ./install
```

## Inside

- **Shell:** zsh + sheldon + starship
- **Editor:** Neovim (lazy.nvim)
- **Terminal:** Ghostty + tmux
- **WM:** Aerospace + sketchybar + skhd
- **Keyboard:** Karabiner (Hyper key, Tab→tmux prefix)

## Layout

- `install` — bootstrap
- `dev-env` — symlink dotfiles
- `runs/` — install scripts
- `env/` — configs
- `scripts/` — utilities

See [AGENTS.md](AGENTS.md) for deeper guidance.
